# Getting Started

Welcome to the getting started guide!

## Installation

First, install the plugin:

```bash
pip install mkdocs-easylinks-plugin
```

## Configuration

Add it to your `mkdocs.yml`:

```yaml
plugins:
  - search
  - easylinks
```

For more details, see the [configuration reference](configuration.md).

## Your First Link

Now you can create links using just filenames:

```markdown
[See the API docs](api.md)
```
