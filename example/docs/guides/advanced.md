# Advanced Usage

This guide covers advanced features of the EasyLinks plugin.

## Configuration Options

See the [configuration reference](configuration.md) for all available options.

## Handling Ambiguous Files

If you have multiple files with the same name, check the [API documentation](api.md) for details on how the plugin handles this.

## Tips and Tricks

### Using Anchors

You can still use anchors with easy links:

```markdown
[Jump to section](getting-started.md#installation)
```

### External Links

External links work as normal and are not processed:

```markdown
[MkDocs](https://www.mkdocs.org)
```
